1. Tag
    Tags are enclosed in angle brackets, usually comes in pairs( opening tag and closing tag)
  2. Element
    The structure from opening tag to closing tag
  3. Head
    Meta Information about the document
  4. Title
    Used to define title of HTML document.
  5. Meta
    Provides information about the document
  6. Link
    Used to link external resources like stylesheets ..etc
  7. b
    It is used to make the text bold
  8. em
    Used for emphasizing the text
  9. u
    Used to underline the text
  10. sub
    Used for creating subscript text (ex : chemical formulas)
  11. anchor
    Used for creating hyperlink
